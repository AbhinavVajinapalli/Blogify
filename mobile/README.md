# Mobile (Flutter)

This folder is reserved for the Blogify Flutter client.

Suggested organization:
- `lib/features/` for feature modules (auth, feed, profile, dashboard).
- `lib/models/` for domain models.
- `lib/widgets/` for reusable UI components.

When the mobile client is implemented, add:
- `pubspec.yaml`
- `lib/main.dart`
- feature-specific state management and service layers.
